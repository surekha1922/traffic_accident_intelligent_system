# 🚦 Traffic Accident Intelligence System

## 📌 Overview
This project predicts traffic accident severity and identifies accident-prone zones using machine learning and geospatial analysis. It also includes a safe route recommendation system based on accident risk.

---

## 🚀 Features
- 🔮 Accident Severity Prediction (ML Model ~70% accuracy)
- 🗺️ Hotspot Detection using DBSCAN clustering
- 🌡️ Weather & Time-based Feature Engineering
- 🔥 Heatmap Visualization (Folium)
- 🚦 Safest Route Suggestion (Dijkstra Algorithm with Risk)

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Folium (Maps)
- NetworkX (Graph Algorithms)

---

## 📊 Results
- Achieved ~70% accuracy in predicting accident severity
- Identified high-risk accident zones using clustering
- Built a risk-aware route system combining distance and accident probability
